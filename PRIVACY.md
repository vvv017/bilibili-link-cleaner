# Privacy Policy

Last updated: 2026-05-16

Bili Share Link Cleaner is a single-purpose browser extension that cleans Bilibili video share links copied from the Bilibili website.

## Data Collection

This extension does not collect, store, sell, transmit, or share any personal data.

## Clipboard Handling

The extension runs only on `bilibili.com` pages. When Bilibili writes a copied share link to your clipboard, the extension rewrites that clipboard text locally in the browser so that tracking parameters and share title text are removed.

The clipboard content is not sent to any server.

## Network Requests

This extension does not make network requests and does not load remote code.

## Permissions

The extension uses a content script limited to Bilibili pages so it can clean copied Bilibili share links at the moment they are copied.

## Contact

For issues or questions, use the GitHub repository issue tracker after the project is published.
